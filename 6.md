이제 기본 레이아웃 작업을 마무리해보자.

css 정리 좀 해보려고 한다.

```css
/* 보면 페이지에 쓸데없는 간격이 있었을 것이다.
  그것을 없에자.
 */
body{
  margin:0px;
  padding:0px;
}

...
/* 네비게이션처럼 포스트도 반응형으로 만들어보자. */
.post{
  max-width: 600px;
  margin:auto;
}
...
```
